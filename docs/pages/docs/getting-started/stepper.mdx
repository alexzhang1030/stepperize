# Stepper Component

The `Stepper` component is a versatile and customizable component that allows you to create step-based workflows in your React application

Internally it works with the `Context` API so we can use the `useStepper` hook in any child component.

## Usage

Here is an example of how to use the `Stepper` component:

```tsx
import { Stepper, defineSteps } from '@stepperize/react';

const steps = defineSteps(
  { id: 'step-1', title: 'Step 1', description: 'Description for step 1' },
  { id: 'step-2', title: 'Step 2', description: 'Description for step 2' },
  { id: 'step-3', title: 'Step 3', description: 'Description for step 3' },
);

const MyStepperComponent = () => {
  return (
    <Stepper steps={steps}>
      {/* Stepper content */}
    </Stepper>
  );
};
```
